# Scripts

This folder contains the exact MATLAB scripts selected for supplementary material release.

## Files

- `statistical_analysis.m`: Multi-group statistical workflow with report and figure export.
- `larvae_hatching_analysis.m`: Mixed-effects model workflow for larvae hatching data.
- `muse_file_cleaner_advanced.m`: Data cleaning/preprocessing utility for Muse exports.

## Running scripts

Run scripts directly in MATLAB and follow interactive dialogs/prompts.
